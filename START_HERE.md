## 🔧 Troubleshooting

### **"Python not found" Error**
Solution:
```powershell
# Install Python from python.org or use:
C:\Python311\python.exe LAUNCHER.py
```

### **"File not found" Error**
Solution:
- Make sure you're in the correct folder
- Check all fixed files exist (`*_FIXED_*.py`)

### **Menu doesn't appear**
Solution:
```powershell
# Run directly with Python
python LAUNCHER.py
```

### **Programs ask for file path**
Solution:
- Use existing file: `C:\Windows\System32\drivers\etc\hosts`
- Or: `C:\\path\\to\\anyfile.txt`
- Or create test: `echo test > C:\temp\test.txt`
