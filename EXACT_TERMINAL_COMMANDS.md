## IF SOMETHING GOES WRONG

**"Python not found"**
```
Solution: Run this first:
python --version
```

**"File not found"**
```
Solution: Run this:
dir
(Check if files exist)
```

**"Command doesn't work"**
```
Solution: Make sure you're in correct folder:
cd
(Shows current path, should end with "digital evidence freamwork")
```

**"Want to stop program"**
```
Press: Ctrl + C
```

**"Want to clear screen"**
```
Type: cls
Press: Enter
```
